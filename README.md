## サイト概要
ランキングを作成や参加して、コメントを楽しむものです。

競い合いから話のネタ作りまで自由に利用してください。

## 機能一覧
- ユーザー名とパスワードによる登録・ログイン
- Twitterログイン
- マイページ
- ランキングページの作成
- ランキングの参加と投稿
- ランキングページ(sort, 偏差値算出)
- コメント欄(非同期通信)
